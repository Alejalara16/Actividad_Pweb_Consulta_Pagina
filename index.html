INFORMACIÓN SOBRE HTML

Son siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar el contenido en la web. Es el lenguaje que se utiliza para definir la estructura básica de una página web, incluyendo elementos como encabezados, párrafos, listas, enlaces, imágenes, formularios y otros tipos de contenido.

CARACTERÍSTICAS DEL HTML

  1. Lenguaje de marcado: HTML utiliza "etiquetas" (tags) para marcar diferentes tipos de contenido.

  2. Estructura de documentos: HTML organiza el contenido en una jerarquía de elementos, donde cada elemento está definido por una etiqueta de apertura y, generalmente, una etiqueta de cierre.

  3. No es un lenguaje de programación: HTML no es un lenguaje de programación en el sentido tradicional; no tiene lógica ni variables. Simplemente define la estructura y presentación del contenido.

  4. Interactividad y estilos: HTML se combina con CSS (Cascading Style Sheets) para estilizar y con JavaScript para agregar interactividad a las páginas web.

  5. Es la base de la web: Todo el contenido en la web, desde sitios simples hasta aplicaciones web complejas, está construido sobre HTML.
